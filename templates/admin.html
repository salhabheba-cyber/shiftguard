📋 Block Sites on ALL Employee Phones (Router Setup)

To block websites on ALL phones connected to spa WiFi:

  1. Open browser → go to your router IP
  2. Login with router admin credentials (usually on sticker on router)
  3. Find DNS Settings or DHCP Settings
  4. Set Primary DNS to: — (this computer)
  5. Set Secondary DNS to: 208.67.220.220
  6. Save and reboot router
  7. ✅ All phones on WiFi will now be blocked from the sites in your list above
⚠️ Notes:
  • This PC must stay ON for the DNS blocking to work on other devices
  • Employees can bypass by switching to mobile data (4G/5G) — that's on their own bill
  • "Apply Blocking on This PC" only blocks on THIS computer, not employee phones
  • Router DNS change blocks ALL devices on that WiFi network
